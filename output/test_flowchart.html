Flowchart of the Pre-Processing Steps

Description
ID: AN001
Analysis Description
Using Ophtha test data for an example preprocessing. Tabular representation of absolute value and change from Baseline in Volume Hard Exudates (unit: nano liters), in the 3 mm Center using the study eye. Reporting values at visits: Baseline, Week 1, Week 4, Week 8. Stratification of patients by: Baseline Volume Hard Exudates Tertile in the 3 mm Center. Volume of Hard Exudates is calculated as the sum of HEV plus IHRMV.
Used Datasets
Test dataset for dummy Study 1.
Test dataset for dummy Study 2.
Used Variables
For Filtering:
AFEYE: Treatment Eye
OEIMDPTH: Image Depth
ANL02FL_97: Analysis Flag 02 at 97 BScan
OELOCDTL: Location Detail
AVISIT: Analysis Visit
Generated Variables
Change from Baseline for Volume Hard Exudates.
SUMV: Volume Hard Exudates: SUMV = IHRMV + HEV
HEV: Intraretinal Hard Exudates Volume.
IHRMV: Intraretinal Hyper Ref Material Volume.
Result of Preprocessing
Dataframe with variables UNI_ID, AVISIT, SUMV, CHG_SUMV, BL_TERTILE_PCT
